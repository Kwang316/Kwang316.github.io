Usage — Slicer Python Console

# 1. Load CT in Slicer: File → Add DICOM Data → your folder
# 2. Open Python console: View → Python Interactor

exec(open('/path/to/volume_renderer.py').read())

# Apply a preset
apply_preset('chest_full')   # or 'bone' | 'vessels' | 'soft_tissue'

# Export 360° turntable GIF
export_turntable('./chest_full.gif', n_frames=72, fps=24)

# Export all 4 presets as PNGs for portfolio
export_all_presets('./renders')
