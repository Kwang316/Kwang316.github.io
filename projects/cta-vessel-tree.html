CTA-Vessel-Tree

Aorta and coronary artery extraction from CTA using VMTK. Outputs centerline metrics and interactive 3D polydata JSON for browser rendering.

Dataset

ASOCA 2020

40 CTA scans with annotated coronary trees

Metrics

Centerline Analysis

Diameter, length, tortuosity index per branch

Output

3D JSON + VTP

Browser-renderable vessel mesh via Three.js

Extraction Pipeline

01CTA DICOM → 3D Slicer volume → threshold-based vessel mask
↓
02VMTK vmtkMeshGenerator → surface mesh from label
↓
03vmtkCenterlines → centerline polydata with Voronoi diagram
↓
04Branch splitting → per-branch diameter / length / tortuosity
↓
05VTP export → convert to glTF/JSON for Three.js viewer

Metrics Export

# centerline_metrics.py
import vmtk, vtk, json

def compute_metrics(centerline_polydata):
    branches = split_branches(centerline_polydata)
    metrics = []
    for branch in branches:
        metrics.append({
            "length_mm":      compute_length(branch),
            "mean_diameter_mm": compute_diameter(branch),
            "tortuosity":      compute_tortuosity(branch),
        })
    return json.dumps(metrics, indent=2)
