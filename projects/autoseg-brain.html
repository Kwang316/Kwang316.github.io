AutoSeg-Brain

Automated brain tumor segmentation pipeline integrating MONAI inference with 3D Slicer for clinical visualization and PDF report export.

Dataset

BraTS 2023

1,251 multimodal MRI cases (T1, T1Gd, T2, FLAIR)

Task

3-class Segmentation

Enhancing tumor, tumor core, whole tumor

Result

Dice ≥ 0.87

On BraTS validation holdout set

Pipeline Overview

01Load DICOM series → convert to NIfTI (dcm2niix)
↓
02Pre-process: N4 bias correction, z-score normalization
↓
03MONAI SegResNet inference → 3-channel probability map
↓
04Argmax → label NIfTI → load into 3D Slicer as segmentation node
↓
05Export: 3D rendering screenshot + volume stats PDF
